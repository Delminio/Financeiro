Financeiro Corporativo Premium v3

Como usar:
1. Extraia o ZIP.
2. Abra o arquivo index.html no navegador.
3. Cadastre primeiro as pessoas.
4. Depois lance recebimentos e contas vinculando cada item a uma pessoa.

Recursos:
- Pessoas responsáveis por salários e contas
- Parcelas automáticas
- Contas recorrentes infinitas
- Alertas de vencimento
- Comprovantes em imagem/PDF
- Exportação CSV e PDF
- Backup e restauração em JSON

Observação:
- Os dados ficam salvos no navegador.
- Se ainda aparecer dado antigo de versões anteriores, use o botão "Apagar tudo".
- Gráficos e PDF usam bibliotecas online, então funcionam melhor com internet ligada.
